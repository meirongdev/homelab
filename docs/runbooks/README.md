# Runbooks

> Reproducible operational procedures.
> Last updated: 2026-03-07

## Available Runbooks

1. `dns-network-failure-recovery.md`: DNS/网络故障恢复（优先）
2. `kopia-backup.md`: Kopia 备份与恢复操作

## Runbook Standard

1. 目标明确：说明触发条件与成功判定。
2. 命令可执行：每一步给出完整命令与执行目录。
3. 可回滚：关键步骤必须有回滚路径。
4. 事后复盘：在对应 `../plans/` 增补时间线与根因。
